# ✅ To-Do List App

A modern and responsive **To-Do List Application** built using **HTML5, CSS3, and JavaScript**. This project demonstrates DOM manipulation, Local Storage, task management, and responsive web design.

This project was developed as part of a **JavaScript Mini Project Internship**.

---

# 📌 Project Overview

The To-Do List App helps users organize and manage their daily tasks efficiently. It provides an easy-to-use interface where users can add, edit, complete, search, and delete tasks. All tasks are stored using the browser's Local Storage, so they remain available even after refreshing the page.

---

# ✨ Features

- Add New Tasks
- Edit Existing Tasks
- Delete Individual Tasks
- Mark Tasks as Completed
- Search Tasks
- Clear Completed Tasks
- Clear All Tasks
- Task Statistics
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- Local Storage Support
- Responsive Design
- Modern User Interface

# 📖 How to Use

### Add a Task

- Enter a task in the input field.
- Click **Add** or press **Enter**.

---

### Complete a Task

Click the **✓** button.

Completed tasks will be marked with a strikethrough.

---

### Edit a Task

Click the **✏ Edit** button.

Update the task and save the changes.

---

### Delete a Task

Click the **🗑 Delete** button.

The selected task will be removed.

---

### Search Tasks

Use the search box to filter tasks instantly.

---

### Clear Completed Tasks

Removes only completed tasks.

---

### Clear All Tasks

Deletes all tasks from the application.

---

# 💾 Local Storage

All tasks are automatically saved in the browser using **Local Storage**.

Your tasks will remain available even after refreshing or reopening the browser.
